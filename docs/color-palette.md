# Marvelous Dreamer — Color Palette

The app uses a custom dark theme based on deep purple backgrounds with a **violet** primary brand color and **emerald green** as accent. Dynamic color is disabled to ensure consistent branding across all devices and Android versions.

---

## Backgrounds

| Token | Hex | Usage |
|---|---|---|
| `BgDeep` | `#0A0710` | Darkest layer, used for depth effects |
| `BgBase` | `#100D1A` | Main screen background |
| `CardSurface` | `#1A1528` | Cards, tab bars, bottom navigation bar |
| `BgElevated` | `#231D35` | Dropdowns, elevated surfaces |
| `BgOutline` | `#2E2647` | Borders, dividers, progress track |

## Purple — Primary Brand

| Token | Hex | Usage |
|---|---|---|
| `Violet` | `#7C3AED` | Buttons, tab indicator, badge backgrounds |
| `VioletLight` | `#9F67F5` | Tab labels, budget values, section headers |
| `VioletSoft` | `#B794F6` | Secondary text with purple tint |

## Green — Accent

| Token | Hex | Usage |
|---|---|---|
| `Emerald` | `#10B981` | FAB, FREE badge, spending remaining |
| `EmeraldLight` | `#34D399` | Active bottom bar item, navigation arrows, explorer badge |
| `EmeraldSoft` | `#6EE7B7` | Muted green for subtle highlights |

## Gradient

The purple → indigo → green gradient is used across hero banners, trip card accent bars, avatar backgrounds and progress indicators.

| Token | Hex |
|---|---|
| `GradStart` | `#6D28D9` |
| `GradMid` | `#4F46E5` |
| `GradEnd` | `#059669` |

## Semantic

| Token | Hex | Usage |
|---|---|---|
| `Amber` | `#F59E0B` | ★ TOP badge, warning states, spent budget |
| `Rose` | `#EF4444` | Delete actions, log out, error states |

## Text

| Token | Hex | Usage |
|---|---|---|
| `Snow` | `#FFFFFF` | Primary text, titles |
| `Mist` | `#D4BBFF` | Secondary text with purple tint |
| `Fog` | `#6B5F8A` | Disabled text, captions, inactive icons |
